# Eldritch Theme for Obsidian

A dark theme for [Obsidian](https://obsidian.md) based on the [Eldritch](https://github.com/eldritch-theme) color palette. Inspired by Lovecraftian horror with deep oceanic tones.

## Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| Background | `#212337` | Primary background |
| Purple | `#a48cf2` | Keywords, blockquotes, H3 |
| Cyan | `#04d1f9` | Accent, links, H1 |
| Green | `#37f499` | Strings, tags, H2 |
| Yellow | `#f1fc79` | Highlights, H6 |
| Orange | `#f7c67f` | Numbers, bold text, H4 |
| Red | `#f16c75` | Errors, important |
| Pink | `#f265b5` | Functions, H5 |

## Installation

### From Obsidian Community Themes

1. Open Obsidian Settings
2. Go to **Appearance** > **Themes**
3. Click **Manage** and search for "Eldritch"
4. Click **Install and use**

### Manual Installation

1. Download `theme.css` and `manifest.json`
2. Create a folder called `Eldritch` in your vault's `.obsidian/themes/` directory
3. Place both files inside the folder
4. In Obsidian, go to **Settings** > **Appearance** and select "Eldritch"

## Features

- Full Eldritch color palette integration
- Syntax highlighting for code blocks
- Styled callouts (info, warning, danger, success, tip)
- Custom heading colors (each heading level has a unique accent color)
- Canvas node styling
- Graph view colors
- Sharp, minimal UI (zero border radius)
